Operating Systems Terminology
TERM DEFINITION
operating system the software that supports a computer's basic functions, such as scheduling tasks, executing applications, and controlling peripherals.
job queue a list of tasks to be carried out in turn by an operating system.
FIFO First In First Out
interactive processing process of interacting directly with a program and/or data for immediately modifying, retrieving and/or displaying information.
time-sharing the operation of a computer system by several users for different operations at the same time.
multiprogramming the running of two or more programs or sequences of instructions simultaneously by a computer with more than one central processor.
application software software that performs specific functions for a user.
system software type of computer program that is designed to run a computer's hardware and application programs
utility software components that assist the operating system to optimize, maintain, organize and manage how it functions
graphical user interface(GUI) a visual way of interacting with a computer using items such as windows, icons, and menus, used by most modern operating systems.
shell a computer program that exposes an operating system's services to a human user or other programs
kernel a core component of an operating system and serves as the main interface between the computer's physical hardware and the processes running on it
device driver kind of software program that controls a specific hardware device attached to a computer
file manager a computer program that provides a user interface to manage files and folders
memory manager the process of controlling and coordinating a computer's main memory
virtual memory a memory management technique where secondary memory can be used as if it were a part of the main memory
boot strapping the process of starting a computer, specifically with regard to starting its software
ROM a type of computer storage containing non-volatile, permanent data that, normally, can only be read, not written to
deadlock situation in which two computer programs sharing the same resource are effectively preventing each other from accessing the resource, resulting in both programs ceasing to function
spooling a process in which data is temporarily held to be used and executed by a device, program or the system
super user (or administrator) a user of a computer system with special privileges needed to administer and maintain the system; a system administrator
multi-factor authentication An authentication system that requires more than one distinct authentication factor for successful authentication
